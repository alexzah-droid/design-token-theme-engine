

# Theme Engine — Usage Guide

## 🎯 Purpose

This document explains how to use themes generated by the Theme Engine in external projects.

---

## 📦 What you get

After build, the /dist folder contains:

- corporate.css
- minimal.css
- apple.css

Each file contains CSS variables wrapped in:

[data-theme="themeName"]

---

## 🔌 How to connect a theme

### 1. Copy files

Copy required theme files from:

/dist

into your project.

---

### 2. Include CSS

Add to your HTML:

<link rel="stylesheet" href="corporate.css">
<link rel="stylesheet" href="minimal.css">
<link rel="stylesheet" href="apple.css">

---

### 3. Add styles

Also include your component styles:

<link rel="stylesheet" href="components.css">

---

### 4. Set theme

Set theme via attribute:

<html data-theme="corporate">

---

### 5. Switch theme (optional)

Example:

<script>
function setTheme(name) {
  document.documentElement.setAttribute("data-theme", name);
}
</script>

---

## 🧩 Example

<html data-theme="apple">
  <button class="button">Button</button>
</html>

---

## ⚠️ Important

- Components must use CSS variables
- Do not hardcode styles
- Themes must remain independent

---

## 🧠 Key Idea

Themes = values  
Styles = implementation  

---

## 📌 Notes

This system works with:

- Web apps
- Google Apps Script (GAS)
- Internal tools
